How to Create a Website for Free with GitHub

Step 1: Sign Up for GitHub

GitHub offers free hosting for static websites. Sign up for an account if you don't have one already.

Step 2: Create a New Repository

Once logged in, click on the "+" icon in the top-right corner and select "New repository".

Name your repository as <your-username>.github.io. This naming convention is necessary for GitHub to recognize it as a user or organization page.

Step 3: Add Website Files

Create HTML and CSS files for your website. These will be the files GitHub hosts for your site.

Step 4: Write Your Website Code

In the HTML file, structure your content using HTML tags such as <header>, <nav>, <main>, <footer>, etc. Style your website with CSS in a separate file.

Step 5: Commit and Push Your Changes

Using Git, initialize a new repository, add your files, commit them, and push to GitHub. Make sure to link your local repository to your GitHub repository.

Step 6: View Your Website

Once your changes are pushed to GitHub, visit https://<your-username>.github.io in your web browser. Your website should now be live!

Step 7: Further Customization

Continue customizing your website by adding more pages, tweaking the design, or adding additional features. GitHub Pages supports Jekyll, a static site generator, for more advanced customization.
